Среды для разработки

Гибкая IDE, одинаково хорошая работающая на разных платформах. Вам определенно стоит остановиться на этой среде разработки.Поддержка следующих языков:C;C++;Fortran.
Среда разработки, предназначенная специально для работы с языком программирования С,но поддерживает и другие:C;C++;Java;Perl; PHP;Python;Ruby.
Cреда является лучшей для тех, кто предпочитает программировать на Python. Работает под управлением Windows, Linux и macOS. Без особых проблем совмещается с Git, Mercurial и SVN.
Поддержка языков:Perl;PHP;Python; Ruby;Tcl.Стабильно и без нареканий работает на macOS, Linux и Windows.Вместе с этой IDE вы получите одноступенчатую отладку, автоматическое заполнение, проверку и подсветку синтаксиса.
Поддерживаемые языки: Java, JavaScript, PHP, Python, Ruby, Go, SQL, С/С++, Groovy, Kotlin. IDE доступна для Windows, macOS, Linux. Одной из особенностей IntelliJ IDEA считается встроенная интеграция с Git, есть AIR Mobile с поддержкой Android и iOS.

Универсальная среда разработки с открытым кодом от Apache. Основной язык – Java, остальные добавляются путем установки расширений. Поддерживает операционные системы: Windows, macOS.
